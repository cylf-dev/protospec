# Runtime Specifications

Specifications for how codecs are executed and distributed.

- [Codec Contract](01_codec_contract.md) — the interface contract between the
  pipeline engine and codec implementations: WIT interface (Component Model),
  binary ABI (Core Wasm), native interface, and signature format
- [Distribution & Registry](02_distribution.md) — how codec artifacts are
  distributed: HTTPS, OCI registries, the warg ecosystem, native plugins, and
  embedded codecs
